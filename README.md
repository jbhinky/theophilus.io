
# Theophilus-UDC Docs Site

This site documents the Neuro-Coding Architecture (NCA) and Theophilus UDC prototype.

To run locally:

```bash
npm install
npm run start
```

To build for production (e.g., Netlify):

```bash
npm run build
```
